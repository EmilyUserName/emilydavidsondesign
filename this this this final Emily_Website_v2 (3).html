About me

Builder at heart,
developer by choice.

I don't have an elevator pitch. I have calluses, a paintbrush, a lesson plan, and a keyboard — and I will deploy all of them in the same week without blinking.

I come to web development from a genuinely hands-on background — I've run my own acupuncture practice, done construction work, and built integrations across various tech industries. I formalized it with a full-stack bootcamp, because doing the thing right matters.

Based on the Maine coast, I build full-stack web products independently, from architecture through deployment. I work best when the problem doesn't have an obvious answer.
